Brief Overview

Vote Meter is a Web based electronic voting system for conducting a transparent voting exercise. It records the whole voting process in real time providing accurate and unbiasing result instantly, By providing a comprehensive vote input system, vote recording, user validation, consolidation and tabulation of voting result.

Vote Meter is based on a modular design, which allows it to be easily understood, flexible and user friendly. Our documentation will guide you through understanding and utilizing the various components that are bundled with the Application.

Responsive layout

The Vote Meter Application is designed to be completely responsive across a variety of devices.
